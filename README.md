# creatix-ai
Creatix AI is an AI-powered UI/UX design generator that transforms text prompts, logos, screenshots, and documents into modern website layouts and production-ready frontend code. It provides live preview, code generation, file-based design analysis, and export functionality, helping designers and developers create professional interfaces faster.
